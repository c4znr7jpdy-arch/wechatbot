# 青岛潮汐插件

数据来自自然资源部北海预报减灾中心的青岛海洋预报平台，不需要 API Key。

## 命令

- `/潮汐`：今天青岛沿岸高潮、低潮和当前涨退潮状态
- `/明天潮汐`、`/后天潮汐`：青岛沿岸指定日期预报
- `/潮汐 石老人`：石老人海水浴场下一期官方精确潮时
- `/潮汐 金沙滩 明天`：指定浴场和日期

查询结果默认生成潮汐信息卡片，将当前状态、高潮和低潮分区展示；图片渲染异常时自动降级为文字结果。
潮汐查询仅响应以 `/` 或 `\` 开头的明确命令，普通聊天中出现“涨潮”“退潮”等词不会触发插件。
- `青岛涨潮退潮时间`：不带命令前缀也可识别

浴场支持一浴、二浴、三浴、六浴、石老人、金沙滩、仰口、银沙滩和灵山湾。

官网逐小时接口提供未来七天青岛沿岸潮位，插件通过相邻整点插值估算高潮、低潮分钟；浴场接口直接提供下一期高潮、低潮准确时间。接口无公开稳定性承诺，因此插件内置十分钟缓存和最长六小时的故障缓存。
