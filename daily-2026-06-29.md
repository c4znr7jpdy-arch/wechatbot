## Additional Changes

- Added live group-chat interjection support to `data/plugins/astrbot_plugin_proactive_chat`.
- The bot now has a separate `group_settings.live_interject_settings` config block for active-chat interjections:
  - `enable_live_interject`: default enabled for already configured group proactive sessions.
  - `probability_percent`: default `1`, meaning a 1% chance per ordinary group message.
  - `cooldown_seconds`: default `300`, preventing repeated interjections in busy groups.
  - `prompt`: instructs the LLM to understand recent context and reply to the last message before the interjection.
- Live interjections reuse the existing context builder, LLM provider path, message sender, and conversation archive flow without changing the idle/silence proactive-message counters.
- Added the new config block to session override sanitization and the admin config renderer order.
