# 投放系统-星云和图文部分
- 星云端
  - 菜单与权限
    - 菜单展示
      - [P0] 星云平台营销中心 下新增投放系统 菜单成功展示
      - [P0] 投放管理、资源位管理 二级菜单展示正确
    - 权限控制
      - [P0] 通过账号配置控制新建、查看、编辑、暂停、复制、发布、删除 权限
      - [P0] 无权限用户直接访问url，校验被拦截或提示无权限
      - 管理员角色 
        - 可编辑全部投放
      - 非管理员角色（授权菜单）
        - 只可编辑当前用户创建的投放
  - 投放管理列表
    - 查询条件
      - 按投放名称模糊查询，返回匹配数据
      - 按投放id精准查询，返回唯一数据
      - 按端多选查询，结果取并集
      - 按页面多选查询，结果取并集
      - 按位置多选查询，结果取并集
      - 按类型多选查询，结果取并集
      - 按状态多选查询，结果取并集
      - 按投放时间区间查询，校验有交集的活动均可命中
      - 按权重精准查询，校验结果正确
      - 点击重置，筛选条件全部清空并恢复默认列表
    - 状态判定
      - [P0] 当前时间在投放时间段内且已发布，状态显示进行中
      - [P0] 当前时间大于投放结束时间，状态显示已结束
      - [P0] 当前时间在投放时间内但活动被暂停，状态显示暂停，前端不可见
      - 保存未发布数据，状态显示草稿中
      - 跨零点、边界秒场景下，状态切换正确
    - 列表操作
      - 查看数据 本期置灰，校验点击不可用
      - 进行中活动支持查看、编辑、暂停、复制，按钮显示正确
      - 已结束活动支持查看、复制、删除，且不展示编辑按钮
      - 暂停活动支持查看、编辑、发布、复制
      - 草稿活动支持查看、编辑、发布、复制
      - 删除按钮仅在已结束状态展示
      - [P0] 点击暂停弹窗二次确认，确认后状态变更为暂停，取消后状态不变
      - [P0] 暂停活动再次发布时，若活动时间未结束则发布成功；若已结束则提示投放已结束 并刷新为已结束状态
  - 图文投放配置
    - 基础设置
      - 进入新建页默认类型为图文
      - 类型字段支持选择图文、优惠券、直播、视频，本期仅图文、优惠券
      - 选择端为单选，支持商城APP、商城小程序、智家APP、智家小程序
      - [P0] 未选端前，不允许选择页面和位置，并给出强提示
      - [P0] 页面根据端联动展示，校验首页、注册页、单品页、搜索页、活动页、支付完成页、我的页、购物车页等选项正确
      - 智家小程序新增首页配置，选择端为智家小程序时，页面下拉可选择首页
      - 智家小程序选择首页后，页面配置能力与商城端保持一致
      - [P0] 位置根据页面联动展示，校验顶部弹窗、悬浮弹窗、普通弹窗、全屏弹窗、活动推荐位、瀑布流 选项正确
      - 智家小程序选择首页后，位置下拉支持活动推荐位、瀑布流
      - 智家小程序首页切换不同位置时，其他配置信息与商城端保持一致
      - 投放名称必填，长度不超过48个字符
      - [P0] 投放开始结束时间精确到秒，结束时间必须大于开始时间且大于当前时间
      - 配置权重范围1-9999，为空、非数字、越界值均不允许保存
      - [P0] 发布时校验同端-页面-位置 下进行中的活动权重不可重复
    - 投放设置
      - 触发时机
        - 弹窗类位置支持常驻展示、延迟展示、离开时展示、完成支付、完成注册 时机，且为必填
          - 单品页
            - 常驻展示
            - 延迟展示
            - 离开时展示
          - 注册页
            - 完成注册展示
          - 支付完成页
            - 完成支付展示
          - 购物车页
            - 延迟展示
        - 选择延迟展示 后，默认1秒，可手动调整秒数
        - 未配置延迟秒数时不可发布
        - 位置为瀑布流、活动推荐位 时，触发时机默认为常驻展示 且不可编辑
      - 投放频次
        - 弹窗默认选择固定次数，不允许选不限
        - 固定次数-每人每天上限支持设置每天上限+累计上限
        - 固定次数-每人累计上限支持设置累计展示次数
        - 输入0、负数、非数字、小数、超大值时，校验提示正确
        - 位置为瀑布流、活动推荐位 时，投放频次默认为不限 且不可编辑
      - 商品定向
        - [P0] 仅单品页 展示商品定向配置
        - [P0] 商品定向必填，可下拉选择商品池名称+id
        - [P0] 未选择商品池不可发布
        - 商品池失效、空商品池、无权限商品池时，校验提示正确
        - 单品页需要验证商品定向相关sku
      - 人群定向
        - [P0] 人群定向必填
        - 选择已有人群时，仅支持单选
          - 不限人群
          - 新人
          - 非smart会员
          - smart会员
        - 导入人群包时，上传成功后展示成功导入用户数量
        - 导入文件存在失败数据时，展示失败数量并支持失败明细下载
        - 导入空文件、重复用户、非法user id、超量用户时，校验失败提示
          - 导入仅支持userid
        - 选择CDP 人群包时，仅展示已启用到投放系统的人群包
        - CDP 人群包展示名称+id，选择后保存成功
        - 人群登录状态
          - 未登录
            - 未登录时无userid，对人群包无uerid的人群生效
    - 投放内容
      - 普通弹窗、悬浮弹窗、顶部弹窗、活动推荐位、全屏弹窗支持选择资源位
        - 侧重验证智家端弹窗，h5新增的功能
      - 资源位下拉仅展示当前用户创建、且投放端包含所选端的数据
      - 瀑布流支持选择资源位，并输入阿拉伯数字自定义投放位置
      - 智家小程序-首页-活动推荐位 支持选择资源位
      - 智家小程序-首页-瀑布流 支持选择资源位并关联瀑布流商品池
      - 自定义投放位置输入非数字、空值、负数时，校验失败
      - 仅瀑布流位置展示关联瀑布流商品池 配置
      - 关联商品池后，前端瀑布流推荐范围与商品池一致
    - 展示时间
      - 支持永久展示、自动关闭、手动关闭
      - 选择自动关闭后，可输入关闭秒数
      - 位置为瀑布流、活动推荐位 时，展示时间默认为永久展示且不可编辑
      - 自动关闭秒数输入边界值时，校验前后端处理一致
    - 操作
      - 点击取消，本次编辑数据清空
      - [P0] 点击保存草稿，执行必填校验，保存后状态为草稿
      - [P0] 点击发布，活动生效并在列表展示为进行中或待生效状态
      - 编辑进行中活动时，类型、基础设置、投放设置、投放内容、展示时间 置灰不可编辑
      - [P0] 编辑进行中活动时，仅允许修改人群包和投放频次
      - [P0] 修改人群包后，重新按新范围投放；同时命中新老人群的用户，频次按累计数据计算
      - 复制活动时带入原活动初始化数据，所有字段允许编辑
      - 复制后的新活动拥有新投放id，不覆盖原活动
  - 商品池优化
    - 菜单迁移
      - 入口
        - 原商品池管理 迁移至投放系统-商品池管理 后，菜单可正常访问
        - 历史商品池数据迁移后，校验列表、详情、编辑能力不丢失
    - 商品池列表
      - 查询
        - 按商品池名称模糊查询，校验返回匹配数据
        - 按商品池id精准查询，校验仅返回唯一匹配数据
        - 名称为空、id为空时点击查询，校验返回默认列表
        - 名称包含特殊字符、超长字符时，校验查询稳定且无注入风险
      - 展示
        - 列表默认展示商品池名称、商品池id、创建时间、操作人、操作
        - 新建商品池后，校验自动生成6位自增id
        - 编辑商品池后，校验最后操作人为当前编辑人
        - 点击编辑，校验正确拉起商品池配置页
      - 分页
        - 切换每页条数，校验列表数量变化正确
        - 点击页码跳转，校验跳转页数据正确
        - 查询后再翻页，校验筛选条件保持
    - 商品池新建、编辑
      - 基础数据
        - [P0] 商品池名称必填，空值保存时提示必填
        - 商品池名称输入24字以内可保存，超过24字提示限制
        - 新建时商品池id置灰不可输入，保存后自动生成
        - 编辑时商品池id置灰不可修改
      - 商品明细方式
        - 批量导入
          - 批量导入、分类、标签 三选一，未选择时保存失败并提示
          - 下载模板成功，模板字段与导入规则一致
          - [P0] 导入正确模板文件，校验商品导入成功
          - [P0] 导入缺字段、错字段、空文件、重复商品、非法商品id时，toast 提示导入数据有误
          - 导入失败后，可下载失败明细，明细内容准确
          - 导入大文件、超量商品时，校验处理性能与超限提示
        - 分类
          - 可下拉展示商品四级分类
          - 支持多选分类，保存后商品池命中多个分类商品
          - 分类层级选择后，校验返回商品范围正确
          - 分类为空时保存失败
        - 标签
          - 下拉展示标签名称+id
          - 商品卖点标签支持多选
          - 标签为空时保存失败
          - 标签失效、标签下无商品时，校验提示和保存逻辑符合预期
        - 限制规则
          - 选择分类后，不允许手动增减商品
          - 选择标签后，不允许手动增减商品
          - 分类、标签模式下不展示商品明细列表
      - 操作
        - [P0] 保存时执行必填校验，成功后返回列表并可查询到新建数据
        - 取消时不保存任何变更

    - 编辑投放时变更人群包，发布后按新人群包范围重新投放。
    - 变更人群包后，原命中用户不再命中时，不再返回该投放数据。
    - 变更人群包后，新命中用户进入页面时，可正常获取投放数据。
    - 同一用户同时存在于旧人群包和新人群包时，展示频次按累计值延续，不重新起算。
    - 重复导入用户、重复命中同一人群包时，频率延续，但不能重复插入
    - 人群包为空、人群包失效、人群包被停用或删除时，投放应返回空或按配置失败处理，不能返回脏数据。
    - 编辑人群包名称但不改变包内容时，实际命中范围保持不变。
  - 资源位链路
    - 资源位链路
      - 资源位正常有效时，后端可按配置返回正确资源位内容。
      - 资源位素材更新后，接口返回最新资源位数据，不返回旧缓存内容。
      - 资源位失效、下线、删除后，接口不再返回该资源位数据。
      - 资源位关联端不包含当前端时，不应返回该资源位。
      - 资源位关联商品池或商品内容发生变更后，返回结果同步更新。
      - 资源位为空、资源位素材缺失、资源位挂载异常时，接口返回空或兜底数据，不返回半残数据。
